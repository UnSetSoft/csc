### Expanded container
:::info
 To use the expanded container, it is mandatory to use `kg-container`
:::

```html
<header class="kg__header">
    <nav class="kg__navbar kg-container kg-expanded">
        <ul>
            <li><a href="#">Home</a></li>
            <li><a href="#">Products</a></li>
            <li><a href="#">Contact</a></li>
            <li><a href="#">About</a></li>
        </ul>
    </nav>
</header>
```
